# Private vault for full captures

Public repo holds **redacted** examples only.

Recommended private layout (not in git):

```
%USERPROFILE%\.groklink\vault\
  cases\<case-id>\CASEFILE.json
  history\rf_history.jsonl
  audit\audit_export.jsonl
  metrics\metrics.jsonl
  raw\*.jsonl
```

Never commit COM ports, usernames, full unredacted captures, or secrets to Pitchfork-and-Torch public repos.
